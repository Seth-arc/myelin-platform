# Myelin Primary Build Source

This folder is the consolidated source package for the professional rebuild.

## Structure

- `docs/`
  - `MYELIN_PROFESSIONAL_REBUILD_PRD.md`
  - `MYELIN_PROFESSIONAL_FEATURES.md`
  - `MYELIN_UI_UX_STYLING_GUIDE.md`
  - `MYELIN_SIMULATION_BUILD_SINGLE_SOURCE_SPEC.md`

- `assets/styles/`
  - `myelin-tokens.css` (shared design tokens for light/dark theme and UI consistency)

- `references/product/`
  - `PRODUCT_SPEC.md`
  - `Product Decisions.md`

- `references/simulation/`
  - All simulation specification files from `3D_simulation_work/*.md`

- `references/ui_baseline/`
  - Current UI baseline and prototype files used as implementation references:
    - `ai-study-buddy-mockup.html`
    - `index.html`
    - `about.html`
    - `login.html`
    - `registration.html`
    - `admin-baseline.html`
    - `lecturer-baseline.html`
    - `module-foundation.html`
    - `module-1.html`
    - `module-2.html`
    - `module-3.html`
    - `module-4.html`
    - `emails.html`

## Notes

- This package is intended to be the primary source of truth for build planning and execution.
- It intentionally excludes large media binaries and non-essential assets to keep the source package focused.
